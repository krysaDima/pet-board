ПРОСТОЙ ЗАПУСК (без знания фронтенда)
=====================================

1) Установите Node.js один раз:
   Откройте в браузере: https://nodejs.org/
   Скачайте версию LTS и установите (галочки по умолчанию — ок).

2) Запустите проект двойным щелчком по файлу:
   запуск.bat
   (он лежит в этой же папке, что и этот текст)

ВАЖНО: не открывайте файл index.html двойным щелчком из проводника —
страница будет пустой. Нужен только адрес из терминала, с цифрой 3 в конце:
http://localhost:5173/
(это не http://localhost:517 — там другой порт и сайт не откроется.)

После запуска запуск.bat браузер на ПК может открыться сам.

Доступ с телефона (та же Wi-Fi, что и компьютер):
  - В чёрном окне найдите строку «Network» с адресом вида http://192.168.x.x:5173/
  - Откройте этот адрес в браузере телефона (не localhost — для телефона это сам телефон).
  - Если не грузится: в Windows при первом запуске разрешите «частную сеть» для Node.js
    в запросе брандмауэра; телефон и ПК должны быть в одной сети.

Если окно сразу закрылось или написало, что npm не найден — значит, Node.js
ещё не установлен или нужно перезагрузить компьютер после установки.

Весь код сайта уже лежит в этой папке — программировать вручную не нужно.
